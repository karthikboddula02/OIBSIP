# BMI Calculator

## Objective
A Python GUI application to calculate Body Mass Index (BMI).

## Features
- Calculate BMI
- Display BMI Category
- User-friendly Tkinter GUI
- SQLite Database Storage
- BMI History
- BMI Progress Graph

## Technologies Used
- Python
- Tkinter
- SQLite3
- Matplotlib

## How to Run

```bash
pip install -r requirements.txt
python main.py
